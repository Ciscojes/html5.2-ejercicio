HTML5 es el lenguaje de marcado utilizado para crear y estructurar páginas web modernas. Gracias a sus diferentes etiquetas, es posible organizar contenido como títulos, párrafos, imágenes, tablas, listas, enlaces, formularios y contenido multimedia de una manera clara y semántica.
